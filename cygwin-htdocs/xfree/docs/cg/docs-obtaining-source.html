Obtaining the Source Code

Source of latest cygwin-x-doc release

To obtain the source of the latest release of the cygwin-x-doc package start the cygwin setup program, and select the package cygwin-x-doc from the category X11. Mark the checkbox labelled src and install. This will install the documentation source in /usr/src/cygwin-x-doc.

Source from git

The documentation source code is available from sourceware.org git. To obtain it, please use the following commands: 
$ git clone git://sourceware.org/git/cygwin-apps/xorg-doc.git
[output as files are checked out]
$ (cd xorg-doc && autoreconf)
[output as autoconfiguration scripts are regenerated]

You should now have the sources in a directory called xorg-doc.

If you just want to look at the Cygwin/X documentation source, use the web interface to the Cygwin/X documentation tree.
