Session type

On this dialog select the session type you want.

Start no client

This will just start the X server. You will be able to start X clients later.

Select this option if you don't understand the others.

Start a program

This will start one local or remote program which will automatically run with the X server. Remote programs are started using SSH.

Open session via XDMCP

This will start a remote XDMCP session. Starting local clients later is limited.

This option is not available in the 'Multiple windows' mode.
