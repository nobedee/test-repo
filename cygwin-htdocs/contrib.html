How can I help out?

This project is entirely dependent on volunteer time. Volunteer time is always in short supply, so if you are thinking about contributing, rest assured that your efforts will be welcome and appreciated.

Help with the following is always welcome:

  • The Cygwin DLL, the UNIX emulation engine itself.
  • Various Cygwin-specific and Windows-integration utilities: cygwin-apps.
  • Maintaining existing and adding new packages in the Cygwin distribution.
  • This website.
  • Our package tools: setup and cygport.

If you are looking to donate money to support our efforts, then check out the donations page.
