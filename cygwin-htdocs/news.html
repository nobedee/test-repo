Cygwin Newsgroups

Cygwin mailing lists are mirrored by the gmane.io NNTP gateway. You can use any Newsreader (e.g. Thunderbird, Knode, Tin) to access the Cygwin mailing lists.

Please note that the gmane website and its newsgroup search interface is down since August 2016. Only the aforementioned NNTP gateway is still up.

Gmane Cygwin newsgroups always start with the prefix gmane.os.cygwin. Here's a list of the active Cygwin mailing lists and their Gmane newsgroup counterparts:

  • gmane.os.cygwin mirrors cygwin
  • gmane.os.cygwin.announce mirrors cygwin-announce
  • gmane.os.cygwin.applications mirrors cygwin-apps
  • gmane.os.cygwin.devel mirrors cygwin-developers
  • gmane.os.cygwin.patches mirrors cygwin-patches
  • gmane.os.cygwin.cvs mirrors cygwin-cvs
  • gmane.os.cygwin.cvs.apps mirrors cygwin-apps-cvs
  • gmane.os.cygwin.talk mirrors cygwin-talk

There are a few more Newsgroups/Mailinglists, but they are old and unused.
